# User Manual - Medical Billing System

## Getting Started

1. Open http://localhost:8086 in your browser
2. Login with your credentials
3. You will be redirected to the Dashboard

## Dashboard

The dashboard shows:
- Today's sales and profit
- Medicine stock statistics
- Low stock and expired medicine counts
- Quick action buttons
- Sales analytics chart
- Alert notifications
- Recent bills table

## Billing (Sales)

1. Navigate to **Billing** from the sidebar
2. Search medicine by name, barcode, or batch number
3. Click a search result to add to cart
4. Adjust quantities in the cart table
5. Select customer (optional)
6. Choose payment mode: Cash, Card, UPI, or Credit
7. Enter amount paid
8. Click **Generate Bill**
9. Click **Print Receipt** to download PDF invoice

## Medicine Master

1. Go to **Medicines**
2. Click **Add Medicine** to create new entry
3. Fill all required fields (code, name, prices)
4. Use edit/delete buttons for existing medicines

## Purchase Stock

1. Go to **Purchases**
2. Select supplier and purchase date
3. Add medicines with quantity, price, expiry, batch
4. Click **Save Purchase** - inventory updates automatically

## Inventory

View stock levels with filters:
- All medicines
- Low stock alerts
- Out of stock items
- Near expiry (30 days)
- Expired medicines

## Reports

1. Go to **Reports**
2. Select report type and date range
3. Click **Generate**
4. Export to Excel, PDF, or CSV

## Returns

1. Go to **Returns**
2. Select return type (Sales or Purchase)
3. Choose medicine and quantity
4. Enter reason and related sale/purchase ID
5. Click **Process Return**

## Prescriptions

1. Go to **Prescriptions**
2. Select customer
3. Upload prescription image/PDF
4. Add optional notes
5. Click **Upload**

## Settings (Admin/Pharmacist)

- Update shop name, GST, address
- Configure invoice footer
- Upload shop logo
- Backup database

## User Management (Admin only)

- Create new users with roles
- Reset passwords
- Lock/unlock user accounts
- Delete users (except default admin)

## Dark Mode

Click the moon icon in the top navbar to toggle dark mode.

## Notifications

The bell icon shows count of low stock and near-expiry alerts.

## Keyboard Tips

- Use Tab to navigate form fields
- Enter barcode in search field for quick lookup
- Use quick action buttons on dashboard for common tasks
